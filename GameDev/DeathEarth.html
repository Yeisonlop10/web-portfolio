Death Earth
Death Earth

Description

This amazing FPS game includes all the knowledge that I have acquired in Unity. Thaks to the great lessons from Game Institute.com. Some of the main features include the robust state machine that gives the zombies outstanding behaviors such as deplenishing energy over time, hearing, seeing, intelligence to judge possible targets. A modified navmesh area so the characters move properly. Guns and supply inventory using scriptable objects. Animation curves to trigger sounds when the player shoots and also for every animation of the zombie attack and walk. Digital assistant with audio logs from other characters. And more. This project is only for my learning purposes.
